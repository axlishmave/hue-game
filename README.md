# 🎨 Hue Game – Sort the Color Gradient

**Hue Game** is a small experimental puzzle built with HTML, CSS, and vanilla JavaScript.  
Your goal is simple: **rearrange the color tiles so that the gradient transitions smoothly** between four base colors placed in the corners.

This is an early prototype, but the core gameplay is already fully functional.

---

## 🕹️ How to Play

- Four base colors are placed in the corners of the board.  
- The remaining tiles are filled with intermediate hues.  
- Drag and rearrange the tiles to form a correct gradient.  
- When all tiles are in the right position, the puzzle is solved.
